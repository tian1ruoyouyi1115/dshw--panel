### 主要发现（3-5 条）

1. 在双向固定效应模型中，盈利能力（NPR）对资产负债率（Lev）呈显著负向影响，说明样本公司整体上更符合优序融资理论，而非传统权衡理论。
2. 分组回归和交互项结果表明，产权性质会显著调节 NPR 与 Lev 的关系；国有企业样本中的负向效应更强，说明 SOE 在盈利改善后表现出更明显的去杠杆特征。
3. 时变系数模型显示，NPR 对 Lev 的负向作用在样本期内始终存在，但影响强度随年份波动，表明优序融资机制具有时序稳健性，同时受到宏观经济环境变化的影响。
4. 变系数模型和门槛模型表明，企业规模会显著影响 NPR 对 Lev 的边际效应；随着企业规模扩大，这一负向作用总体增强，且在约 27 亿元资产规模附近存在较明显门槛。
5. 引入交互固定效应（IFE）后，NPR 的估计系数仍显著为负，但绝对值较 TWFE 明显缩小，说明优序融资结论具有稳健性，同时传统 TWFE 可能高估了盈利能力对资本结构的影响幅度。
